COMMON GROUND WEBSITE

1. Unzip this folder.
2. Open the folder in Visual Studio Code.
3. Double-click index.html, or use Live Server.
4. Replace "your-email@example.com" in contact.html.
5. Edit the text in any page.
6. To publish free:
   - Create a GitHub account.
   - Create a new public repository.
   - Upload every file from this folder.
   - Open Settings > Pages.
   - Under "Build and deployment," choose "Deploy from a branch."
   - Select main and /root, then Save.
